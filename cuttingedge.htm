Google AutoML

Google AutoML was announced in May 2017. In essence, AutoML is a machine learning software that is building machine learning software itself. In a recent statement by Google, they say it scored 82% in a task about categorizing images. The software is intended to be democratized and freed up for universal use, and could greatly speed up progress in this area.

Find more on https://www.allaboutcircuits.com/news/google-automl-replicate-next-gen-ml-software-more-capable-human-made-system/


Crystalnet reveal

Microsoft recently unveiled Crystalnet at the ACM Annual Symposium on Operating Systems Principles. There, it was described as “a high-fidelity, cloud-scale emulator that helps network engineers nearly eliminate network downtime related to routine maintenance and upgrades as well as software bugs and human errors.” It doesn’t have any plans to be made commercially viable but Microsoft have stated that the interest has made them pursue the perfection of the emulator.

Find more on https://www.theregister.co.uk/2017/11/01/microsoft_crystalnet_network_simulator/


Cryptonite

new tech to obfusicate network topology

"CryptoniteNXT Micro Shield Segmentation significantly reduces attack surfaces accessible via lateral movement.”

The program monitors and blocks malicious activity between a perimeter firewall and internal networks while allegedly preserving performance of said network. The CTO of Cryptonite states: "The network has become an easily infiltrated space that cannot be completely locked down. With CryptoniteNXT, we take away the attackers ability to 'see' within the network, making network attacks or data exfiltration attempts significantly more difficult and expensive for attackers."

Find more on https://www.theregister.co.uk/2017/10/30/cryptonite/
